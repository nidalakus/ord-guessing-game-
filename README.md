# Word Guessing Game

A Wordle-inspired word guessing game written in C.

## How to Play
- Guess a 5-letter word within 6 attempts
- [T] = correct letter, correct position
- [F] = correct letter, wrong position
- [X] = letter not in the word

## Features
- 100+ word database
- Random word selection
- Letter-by-letter feedback system
- Attempt counter

## Language
C
